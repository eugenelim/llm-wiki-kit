# RFCs

Request-For-Comments documents propose changes that touch multiple packages,
affect external users, reverse a prior ADR, or change a top-level convention.
See `docs/CONVENTIONS.md` for the lifecycle. Smaller changes use a normal PR or
a spec (`docs/specs/`).
