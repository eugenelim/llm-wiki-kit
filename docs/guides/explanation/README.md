# Explanation

Understanding-oriented user docs, per [Diátaxis](https://diataxis.fr/).
Conceptual material: why a design works the way it does, what concepts
mean, how systems fit together. The reader wants to understand, not to do
or to look something up.

Empty for now. No existing kit content fits this bucket yet; new
explanatory docs (e.g. "how the journal works", "what a managed region
is") will land here as they're written.
