# Tutorials

Learning-oriented user docs, per [Diátaxis](https://diataxis.fr/). A tutorial
takes a beginner from nothing to a small complete success — step-by-step,
with concrete commands and predictable outputs. The reader follows along
and produces a working result by the end.

Empty for now. The kit's existing user docs still live flat under
`docs/guides/*.md`; gradual migration into the right Diátaxis bucket is a
future docs PR, not adoption scope.

Candidate to migrate here in that future PR:

- `setup.md` — install + first vault. A getting-started walkthrough is the
  canonical tutorial shape.
