# How-to guides

Task-oriented user docs, per [Diátaxis](https://diataxis.fr/). A how-to is
a recipe for solving one specific problem: the reader already knows what
they want and needs the steps to do it. Not for beginners, not exhaustive
— focused.

Empty for now. The kit's existing user docs still live flat under
`docs/guides/*.md`; gradual migration into the right Diátaxis bucket is a
future docs PR, not adoption scope.

Candidates to migrate here in that future PR:

- `customizing.md` — tailoring a vault after `wiki init`.
- `sync-options.md` — picking a sync backend (git, iCloud, Dropbox, etc.).
- `web-clipper.md` — capturing web content into the vault.
