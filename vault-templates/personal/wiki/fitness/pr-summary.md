## Notes

- "Best" reflects the most recent PR; older PRs are preserved in each exercise's `## PR History`.
- e1RM uses the Epley formula (`weight × (1 + reps/30)`) by default; for very low rep counts, treat with caution.
- For body-weight movements (pullups, dips), `weight` represents added weight (or 0 for unweighted).
